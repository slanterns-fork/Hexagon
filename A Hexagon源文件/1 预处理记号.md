## A.1 预处理记号



预处理记号是指向编译器申请各种事项的记号。

它们全部由一个```#```打头，以行末结束。

|        命令名 | 含义                                       |
| ---------: | :--------------------------------------- |
| \#override | 提示下一个出现的方法应当重写超类中继承的方法。如果下一个出现的方法并没有如此做，编译器应当终止编译并汇报这个错误。 |
|  |                                          |
